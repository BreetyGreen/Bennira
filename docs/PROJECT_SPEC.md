# Bennira 项目规范

## 项目身份

- 项目名：`Bennira`
- 大版本命名主题：《八方旅人》系列职业
- 第一个大版本代号：`小偷`
- 当前大版本线：`Bennira 一代 - 小偷`

## 大版本命名规则

从 `1.x` 开始，Bennira 的每一个大版本都使用《八方旅人》系列中的职业名作为版本代号。

命名格式：

- 大版本展示名：`Bennira <序号> - <中文职业名>`
- 里程碑名称：`<序号>.0 <中文职业名>`
- 发布说明标题：`Bennira <序号>：<中文职业名>`
- 技术分支名：优先使用拼音或英文转写，例如 `release/1-xiaotou`

中文职业名是正式版本代号。英文名只作为资料对照，不作为项目版本命名的主名称。

同一个大版本线内，小版本和补丁版本继承该大版本的职业代号。只有进入下一个大版本时才更换职业。

## 选择原则

- 新的大版本优先选择尚未使用过的职业名。
- 基础职业适合常规大版本。
- 隐藏职业适合架构升级、核心能力重做、产品方向明显扩展等关键大版本。
- 如果同一个职业存在不同中文译名，以本文件登记的中文名为准。
- 如需在代码、分支、包名中使用 ASCII 标识，可以使用拼音或英文转写，但对外版本名仍以中文为准。
