# Bennira 路线图

## 当前阶段

当前阶段：`Bennira 一代 - 小偷` 的 **Alpha 已完成**，并已越过原始 Alpha 边界——Bennira 现在是**可运行的真 agent**（可执行 shell、可读写并修改业务代码、支持模型原生 tool_calls），正朝小偷 MVP 推进。

近期已交付（详见 `CHANGELOG.md`）：

- 真实 agent 循环 + 交互式 REPL（真流式、Ctrl+C 中断、slash 命令 + Tab 补全 + 命令历史）。
- 模型原生 tool_calls（`role:tool` 回喂观察），fake-provider 回归网。
- 多服务商 / 多 key 管理，内置模型目录 + 配 key 后实时拉取。
- 终端主题自适应（OSC 11 背景探测 + 色彩深度降级），盗贼紫主题。
- `node --test` 全绿（core 15 个测试文件 + cli 1 个）。

**下一步（进行中）**：apply_patch + diff 预览——改代码前先产出结构化 diff、经确认再落盘，替代整文件覆写。

## 一代：小偷

主题：观察、潜入、小步行动。

核心目标：

### 小偷 Alpha（已完成 ✅）

- CLI 原型。
- 工作区观察。
- 中文需求澄清。
- 文档沉淀。
- 事件日志。
- 会话恢复摘要。

### 小偷 MVP（进行中）

- CLI 原型。
- 项目观察。
- 中文需求澄清。
- 文档沉淀。
- patch 修改。
- Git diff。
- 命令执行。
- 权限确认。
- 事件日志。

完成标志：

### Alpha 完成标志

- 能在本地项目中启动。
- 能读取项目规则。
- 能回答项目当前状态。
- 能把模糊想法整理成项目文档。
- 能输出下一步任务。
- 能记录行动日志。
- 换会话后能根据文档恢复项目状态。

### MVP 完成标志

- 能对小任务提出计划。
- 能生成 patch。
- 能在确认后修改文件。
- 能展示 Git diff。
- 能运行可用验证命令。
- 能总结修改、验证结果和下一步。

## 隐藏职业保留

隐藏职业保留给更大的架构跃迁：

- `发明家`：插件系统、工具构建器、多 Agent 工作流。
- `魔导士`：高级模型编排、跨模型推理。
- `星咏人`：长期记忆、时间线、项目历史推演。
- `武器大师`：复杂代码修改和大型重构。
