# Bennira 路线图

## 当前阶段

当前阶段是 `Bennira 一代 - 小偷` 的需求与架构奠基期。

目标：

- 明确产品定位。
- 完成竞品调研。
- 确定内核架构。
- 定义 Alpha 与 MVP 边界。
- 确保换电脑后可继续。

## 一代：小偷

主题：观察、潜入、小步行动。

核心目标：

### 小偷 Alpha

- CLI 原型。
- 工作区观察。
- 中文需求澄清。
- 文档沉淀。
- 事件日志。
- 会话恢复摘要。

### 小偷 MVP

- CLI 原型。
- 项目观察。
- 中文需求澄清。
- 文档沉淀。
- patch 修改。
- Git diff。
- 命令执行。
- 权限确认。
- 事件日志。

完成标志：

### Alpha 完成标志

- 能在本地项目中启动。
- 能读取项目规则。
- 能回答项目当前状态。
- 能把模糊想法整理成项目文档。
- 能输出下一步任务。
- 能记录行动日志。
- 换会话后能根据文档恢复项目状态。

### MVP 完成标志

- 能对小任务提出计划。
- 能生成 patch。
- 能在确认后修改文件。
- 能展示 Git diff。
- 能运行可用验证命令。
- 能总结修改、验证结果和下一步。

## 二代候选：学者

主题：理解、索引、知识。

可能方向：

- repo map。
- 代码符号索引。
- 语义检索。
- 项目知识库。
- 文档问答。
- 架构解释。

## 三代候选：药师

主题：诊断、修复、健康检查。

可能方向：

- 测试失败诊断。
- lint 自动修复。
- 依赖健康检查。
- 项目风险扫描。
- Bug 复现和修复闭环。

## 四代候选：商人

主题：成本、路由、生态。

可能方向：

- 多模型路由。
- 成本估算。
- 插件分发。
- 供应商切换。
- BYOK。

## 隐藏职业保留

隐藏职业保留给更大的架构跃迁：

- `发明家`：插件系统、工具构建器、多 Agent 工作流。
- `魔导士`：高级模型编排、跨模型推理。
- `星咏人`：长期记忆、时间线、项目历史推演。
- `武器大师`：复杂代码修改和大型重构。

## 当前最近任务

1. 确认产品定位是否成立。
2. 确认“小偷 Alpha”范围。
3. 做一轮不超过三天的定向源码调研。当前已完成第一轮公开源码调研。
4. 决定第一版技术栈和模块边界。当前建议为 TypeScript + Node.js + packages/core + packages/cli。
5. 创建最小 CLI 工程。下一步开始。
6. 实现文件扫描、读取、搜索。
7. 实现事件日志。
8. 实现项目文档生成或更新。
9. 再进入 patch 修改流程。
